# vlsd

Levenshtein String Distance

## Usage

```v
import zztkm.vlsd

distance := vlsd.string_distance("Hello", "Helllo")
```

module doc: [docs/lsd.md](docs/lsd.md)

## Installation

```
v install zztkm.vlsd
```
